FAST: Design Tokens

Design Tokens are reactive CSS custom properties that update all usages automatically.

How it works:

// Create token with default value
const bgColor = DesignToken.create('bg-color').withDefault('#ffffff');

// Use in CSS — maps to --bg-color
css`:host { background: var(--bg-color); }`

// Update token — ALL usages update automatically!
bgColor.setValue('#000000');
